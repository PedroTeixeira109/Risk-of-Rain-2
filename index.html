Captain

About the character

Captain is a character in the game Risk of Rain 2, he is unlocked by beating the final boss of the game and is one of the strongest characters.

Abilities

  1. Passive-Defensive Microbots
  2. Primary-Vulcan Shotgun
  3. Secondary-Power Tazer
  4. Utility-Orbital Probe
  5. Special-Orbital Supply Beacon

About the play Style

Captain can be a short distance character as well as a long distance character with is shotgun that can be charged into a smaller shot that deals the same damage as all the shotguns shots
